Bedrijf

Strandeiland JS

Het probleem

Het design die nu is gemaakt moet gaan werken. Hoe kunnen gebruiker (mensen van Gemeente Amsterdam) het systeem gaan gebruiken?

De oplossing

Een canvas getekende kaart met JavaScript. Met Json moet data worden gestuurd vanuit de database.

Met het gesprek met Michel bij opdracht: Strandeiland Plattegrond vroeg hij: “Hoe zou dit werkend gemaakt kunnen worden?” Ik zei: “Ik heb echt geen idee”. Ik ging deze opdracht bespreken met Tim. Hij stelde voor om dit met Canvas te gaan doen. (Zie Onderzoek documentatie, blz 4 voor Canvas onderzoek).

Wat heb ik geleerd?

Dit is de meest complexe opdracht die ik heb gemaakt. Hierbij heb ik net zoals bij ColorIsDark een hele nieuwe manier van programmeren met Javascript ontdekt. Met de documentatie van de code begrijp ik de code maar ik zou tot nu toe nog niet zelfstandig dit soort code kunnen schrijven. Het was een nieuwe ervaring die ik graag verder wil onderzoeken zodat het mij kan helpen voor het worden van een front-end developer.

Gemaakt werk

  • Regel:
  • 2. Aanroepen van de canvas in de index.html
  • 7. De achtergrond img klaar maken
  • 8. Wanneer de functie gaat beginnen moet de content erin gaan werken.
  • 10 t/m 13. De width en height moet berekend worden van de canvas zodat hij responsive is. De ratio bepaald de berekening van hoe groot je scherm is zodat de tekening klopt.
  • 18. Parameters worden meegegeven aan de functie zodat hij weet welke plaatjes er moeten komen. De anoniemen functie met draw is voor de gebieden die op de canvas komen. De nummers er achter zijn de coördinaten. (Weten waar hij op de canvas moet komen te staan. De coördinaten worden opgenomen in Photoshop omdat de gebieden worden gemaakt met Photoshop. De Anchor is standaard center, maar die moest verandert worden naar rechts boven.
  • 22 t/m 29. Er wordt berekend dat de gebruiker klikt alleen op dat gebied en niet ergens anders. Canvas is 1 groot doek. Het is anders dan HTML CSS.
  • 35. De tekst wordt geplaatst.
  • 46 t/m 53. Wanneer de gebruiker klikt op het gebied, komt de description tevoorschijn.
